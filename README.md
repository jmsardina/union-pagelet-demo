# Union Pagelets

This is to demonstrate how to integrate with the Union pagelets without JSX.

## Development
1. Setup
```bash
$ yarn install
```

2. Bundle
```bash
# You'll need to run this every time you make a change to one of the JS files
# because watch mode is not setup
$ yarn bundle
```

3. Serve
```bash
$ yarn start
```

4. Open
```
$ yarn open
```
